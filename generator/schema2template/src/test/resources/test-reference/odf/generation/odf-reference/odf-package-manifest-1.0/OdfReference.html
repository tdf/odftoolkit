ODF 1.0 (odf-package-manifest) XML Reference

This is an alphabetical ordered list of the XML defined in ODF 1.0 (odf-package-manifest) together with allowed parent, child elements and allowed attributes.

Explanation

  1. Bold blue letters describe a mandatory XML node.
  2. The asterisk * symbol marks child elements which can occur multiple times.
  3. A tailing square brackets with a number within (e.g [1]) marks one of multiple grammar definitions for the same node. For example, manifest:key-derivation-name[1]

manifest:algorithm Element

Parent Elements <manifest:encryption-data>   
Attributes manifest:algorithm-name  manifest:initialisation-vector   
Child Elements  
Child Relations <manifest:algorithm manifest:algorithm-name="#string" manifest:initialisation-vector="#base64Binary" >
