ODF 1.2 (odf-package-manifest) XML Reference

This is an alphabetical ordered list of the XML defined in ODF 1.2 (odf-package-manifest) together with allowed parent, child elements and allowed attributes.

Explanation

  1. Bold blue letters describe a mandatory XML node.
  2. The asterisk * symbol marks child elements which can occur multiple times.
  3. The square brackets with a number within [1] after an XML node like office:value-type[1] mark different grammar definitions.

Here's a shortcut to the content's toplevel element.
