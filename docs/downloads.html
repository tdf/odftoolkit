Current Version

The current version is the 0.10.0 supporting ODF 1.2. The future development is based on the master branch.
The releases of 0.9.0 and 0.10.0 were both on the July 1st, 2021.

Releases (latest)

  1. The release 0.10.0 is using >=JDK 9 implementing ODF 1.2.
    It is providing the new collaboration feature and adding more than half a dozen years of development by merging the ODFDOM fork of Open-XChange's browser office OX Documents.
  2. The release 0.9.0 is using JDK 8 implementing ODF 1.2.
    It is for the final time including the Simple API - a fork once made by IBM but no longer maintained, containing too much copy/pasted functionality. Its purpose was to be the first release of the TDF being close to the latest Apache sources as possible.

For more details see the release notes. People interested should also follow the mail list to track progress.

Prior Apache Releases

Apache ODF Toolkit became an incubator project in August 2011. Prior to that date the ODF Toolkit was a set of projects of ODF Toolkit Union. Now, all of the legacy ODF Toolkit Union releases are archived at apache-extras.org.
